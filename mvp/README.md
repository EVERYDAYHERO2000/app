# Приложение для отслеживания заказов доставки

Мобильное веб-приложение для создания и отслеживания заказов доставки строительных материалов (песок, щебень, асфальт).

## Настройка

### Яндекс.Карты API

Для работы карт необходимо получить API ключ:

1. Перейдите на https://developer.tech.yandex.ru/
2. Создайте новый ключ для JavaScript API
3. Замените `YOUR_API_KEY` в файлах:
   - `create-order.html` (строка 10)
   - `order-detail.html` (строка 10)

## Использование

1. Откройте `index.html` в браузере
2. Нажмите кнопку "+" для создания нового заказа
3. Заполните форму и выберите место доставки на карте
4. Просматривайте детали заказа и отслеживайте доставку

## Функции

- Создание заказов с выбором материала, объема, даты и места доставки
- Просмотр списка всех заказов
- Отслеживание статуса заказа
- Интерактивная карта с выбором места доставки
- Симуляция отслеживания груза в реальном времени

## Технологии

- HTML5
- CSS3 (Material Design)
- JavaScript (Vanilla JS)
- Яндекс.Карты API
- LocalStorage для хранения данных
